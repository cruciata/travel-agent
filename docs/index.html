🌍 智能旅行规划Agent

基于ReAct架构的智能旅行规划系统

根据用户需求自动规划最优路线、查询天气人流、推荐美食

🧠 ReAct架构

🧠
思考 (Thought)
分析问题，确定下一步行动
↓
🔧
行动 (Action)
调用工具获取信息
↓
📊
观察 (Observation)
整合结果，更新知识
↓
🔄
循环
直到完成目标

✨ 核心功能

🗺️

路线优化

贪心算法规划最优游览顺序

🌤️

天气查询

实时天气预报和出行建议

👥

人流预测

景点拥挤程度分析

🍜

美食推荐

智能推荐附近高分餐厅

⏱️

时间规划

精确计算游玩和交通时间

🎨

友好界面

基于Streamlit的美观Web界面

🛠️ 5个专业工具

工具名 功能
get_weather 查询天气预报
get_crowd 查询景点人流
plan_route 规划最优路线
calculate_time 计算游玩时间
recommend_food 推荐附近美食

🚀 快速开始

# 克隆项目
git clone https://github.com/cruciata/travel-agent.git
cd travel-agent

# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
